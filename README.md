# 🏙️ ИИ-Полис «ГЕНОМ» v4.0

Автономная система управления ИИ-производством.

## Архитектура

```
┌─────────────────────────────────────────────────────┐

│          Оркестратор / Диспетчер / Экономист          │
│                                                       │
│  ┌─────────┐   ┌──────────┐   ┌───────────────────┐ │
│  │ Redis   │   │ Unit     │   │ Resource Monitor  │ │
│  │ Bus     │   │ Economy  │   │ (CPU/RAM/Temp)    │ │
│  └────┬────┘   └──────────┘   └───────────────────┘ │
│       │                                               │
│  ┌────▼────────────────────────────────┐             │
│  │        Очереди: CRITICAL → EXPORT   │             │
│  │                 → INTERNAL          │             │
│  └────┬────────────────────────────────┘             │
│       │                                               │
│  ┌────▼────────────────────────────────┐             │
│  │          ЖКХ (3B GPU RX560)         │             │
│  │  🔧 SysAdmin  🔍 Auditor           │             │
│  │  💰 Economist 🧹 Cleaner           │             │
│  │  🚨 MCHS                           │             │
│  └─────────────────────────────────────┘             │

└─────────────────────────────────────────────────────┘
```

## Быстрый старт

```bash
# 1. Установка
bash setup.sh

# 2. Запуск Администрации
python3 -m core.orchestrator

# 3. Запуск Watchdog (в отдельном терминале)
python3 -m security.watchdog
```

## Стек технологий

- **Python 3.12** — ядро Администрации
- **Redis 7** — шина данных (очереди, Pub/Sub, Streams)
- **Ollama + ROCm** — инференс моделей на AMD RX 560
- **ChromaDB** — векторная БД (долгосрочная память)
- **Docker Compose** — оркестрация инфраструктуры

## Роли ЖКХ (Modelfile-костюмы)

| Роль | Описание |
|------|----------|
| SysAdmin | Docker, сеть, файловая система |
| Auditor | Статический анализ, безопасность |
| Economist | Оценка ресурсов в Юнитах |
| Cleaner | Очистка кэшей, логов, мусора |
| MCHS | Аварийное реагирование |
